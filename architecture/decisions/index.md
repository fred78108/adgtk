# Architecture Decision Records

This directory records significant architectural decisions made during the development of ADGTK. Each ADR captures the context, decision, alternatives considered, and consequences.

## Status Key

| Status | Meaning |
|--------|---------|
| **Accepted** | Decision is in effect |
| **Superseded** | Replaced by a later ADR |
| **Deprecated** | No longer applicable |
